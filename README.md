### 🚀 Featured Work

* **[NLP_multihead_multiclass](https://github.com/shivaheidari/NLP_multihead_multiclass)**: A robust PyTorch-based NLP framework tailored for the healthcare domain. It features multi-head, multi-class classification on DICOM text data leveraging models like **BioBERT**. It implements advanced, memory-efficient training strategies using **Hugging Face PEFT (LoRA)** and **QLoRA** with 4-bit/8-bit quantization.
